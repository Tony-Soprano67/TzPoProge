Игра Лабиринт
Цель игры:

1.	Собрать все монеты в лабиринте.
2.	Найти и подобрать ключ.
3.	Дойти до двери, имея при себе ключ
Уровень считается пройденным, когда персонаж касается двери, а ключ уже в инвентаре. Собранные монеты влияют на итоговый счёт.

Функциональные требования

1.	Игровое поле (карта)

•	Карта представляет собой двумерную сетку клеток(матрица) или будет задана через файл.

•	Каждая клетка может быть одним из типов:

i.	Стена (непроходима)

ii.	Пол

iii.	Дверь (проходим если есть ключ).

•	Поверх пола могут располагаться объекты:

i.	Игрок

ii.	Монета

iii.	Ключ

•	Карта задаётся через файл

2.	Управление

•	Реализовать экранные кнопки

i.	Вверх

ii.	Вниз

iii.	Влево

iv.	Вправо

•	Дублировать управление клавишами-стрелками клавиатуры.

•	Кнопки должны быть сгруппированы в крестовину.

•	Каждое нажатие перемещает игрока ровно на 1 клетку в выбранном направлении, если там нет стены.

Правила перемещения и взаимодействия 

1.	Игрок не может пройти сквозь стену.

2.	Если игрок заходит на клетку с монетой она считается собранной и исчезает с карты добавляя к счёту определённое количество очков.

3.	Если игрок заходит на клетку с ключом: 

a.	Ключ считается подобранным.

b.	Он исчезает с карты.

c.	Счетчик ключей изменяется на определённое количество.

4.	Клетка с дверью является стеной до тех пор, пока в счёте ключей значение 0.
